# Hackathon Roadmap

## Day 1

- Finalize project story and architecture.
- Run synthetic demo pipeline.
- Prepare real data schema and sample ingestion notebooks.

## Day 2

- Connect at least one real dataset, preferably CPCB station data or Sentinel-5P HCHO.
- Validate the data alignment by date, latitude, and longitude.
- Generate first AQI and HCHO maps.

## Day 3

- Add model evaluation and visual polish.
- Improve source-region attribution.
- Prepare presentation slides and demo narrative.

## Scoring Strategy

- Show working dashboard first.
- Show scientific methodology second.
- Show real-data extension plan third.
- Emphasize transport-aware HCHO detection as the differentiator.
